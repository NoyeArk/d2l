# Kaggle实战2：树叶分类

## 1 比赛介绍

竞赛地址：https://www.kaggle.com/competitions/classify-leaves/overview

竞赛任务是预测叶子图像的类别。该数据集包含 176 个类别、18353 张训练图像和 8800 张测试图像。每个类别至少有50张图片用于训练。
测试集平均分为公开和私有排行榜。
